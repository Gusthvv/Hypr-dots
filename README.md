## Descriptions:
- **OS:** Arch Linux
- **Window Manager:** Hyprland
- **Bar:** Waybar
- **Notifications:** Mako
- **Laucher:** Rofi
- **Terminal:** Kitty
- **File Manager:** Pcmanfm
- **Fonts:** Nerd Fonts
- **Wallpaper Manager:** Hyprpaper
- **Audio System:** PipeWire
- **Browser:** Chromium
- **Task Manager:** Btop+

## Keybindings

Here are some basic keybindings to get you started.

| Key-bind                | Action                   |
| ----------------------- | ------------------------ |
| **Super + R**           | Application launcher     |
| **Super + E**           | File manager             |
| **Super + Q**           | Open terminal            |
| **Super + F12**         | Power Off                |
| **Super + F11**         | Reboot                   |
| **Super + C**           | Close current window/app |
| **Super + F**           | Open Browser             |
